## How to Compile
Compile using make.

## How to Run
Run the program by piping an input file into the executable:
```
    ./main < input/<input_file>
```

### Example:
```
    ./main < input/example.in
```
Can compare output in terminal to expected example output at 
```
    output/example.out
```
